# Leli Rentals - Admin Dashboard

This is the admin dashboard for managing the Leli Rentals platform.

## Getting Started

1.  Navigate to this directory:
    ```bash
    cd admin.leli.rentals
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

- User management
- Listing approval/moderation
- Verification document review
- System analytics
